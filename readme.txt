Ag Tech is a great theme for headless WordPress.

== Description ==

Ag Tech is a theme that is empty. It doesn't have any posts, menus or widgets to show you. All it has is your title, tagline and the powered by WordPress footer.

== Installation ==

1. Upload the theme files to your themes directory.
2. Go to Appearance > Themes in the dashboard.
3. Activate the Ag Tech theme. Yay!
